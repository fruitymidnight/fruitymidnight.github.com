Github Pages でブログ

Jan 14th, 2013

Github を利用してブログを書くことができる、とのことなのでテスト。

ファイルの管理は git 、記法は markdown を利用できるので、どちらも勉強中の私にはちょうどいいかなと思っています。

以下 markdown の記法を試してみます。

markdownの記法

段落

１つ以上の連続した文字列。空行によって分けられる。 これは別の段落とはみなされない。

これは別の段落になる。

１つ以上の連続した文字列。空行によって分けられる。
これは別の段落とはみなされない。

これは別の段落になる。

文字装飾

文字の強調、下線、/斜線/、

*強調*

コード

コードの記載には文頭に4つ以上の空白をいれる。 文章中にいれるときは\コード“と記載する。

見出し

文章の見出し。

# 見出し１
## 見出し２
### 見出し３

#
